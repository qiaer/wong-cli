### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `wong` 命令了。

- `wong init vue-template myVue`
- `wong config get`
- `wong config set type orgs`
- `wong config set registry vuejs-templates`

- `wong config set type users`
- `wong config set registry YvetteLau`

### 发布

`npm run publish`

开发完成后，即可发布至 npm.
